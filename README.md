# MyTetris
## Introduce
A simple Tetris with pygame   
can show next piece   
can save a piece and use it when need  
## Operation  
|Key|function|  
|:-----:|:-----:|  
| LEFT/RIGHT| let current piece  move to left/right |  
|UP|let current piece rotate|   
|SPACE|swap current piece with saved piece|   
## Requirement
- python3.6
- pygame
- numpy
